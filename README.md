# Kaduna-Primary-Healthcare-Facilities-dashboard
Longitudinal study for Primary Healthcare Facilities in Kaduna state (2025). The dataset is owned by Natview Foundation for Technology Innovation, Kaduna.

Kaduna Primary Health Facilities Dashboard

Overview: This project presents a Power BI dashboard developed to analyze operational, service delivery, and workforce indicators across primary health facilities in Kaduna State.

Objectives: 1. Assess facility structure and service availability  2. Examine workforce distribution and attendance patterns 3. Evaluate vaccine storage capacity and cold chain functionality  4. Identify gaps in preventive healthcare services 5. Support evidence-based decision-making for health system improvement

Tool Used: Power BI

Key Insights: 1. Primary healthcare facilities account for the majority of facilities in the dataset 2. Workforce absenteeism emerged as a major operational challenge 3. Preventive services, particularly cervical cancer screening, showed limited availability 4. Vaccine storage infrastructure was functional in most facilities with storage capacity 5. Community governance structures were widely established across facilities

Data Availability: The underlying dataset contains sensitive health information and is not publicly available. This repository includes dashboard screenshots, project documentation, and summary findings only.
